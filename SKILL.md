# 📊 交易技巧 002 - 固定规则过滤系统 (v2.1)

> **"我有自己的交易系统，不是万能适配器"**  
> 
> **第一性原理**: 市场不存在"适应每一只股票"这种概念  
> 正确做法：**固定规则 → 随机抽样 → 符合就执行 / 不符合就放弃**

---

## 🎯 核心哲学

| ❌ 错误理解 | ✅ 正确理解 |
|-----------|-----------|
| "自我优化 = 每只股票找最优参数" | "自我优化 = 不断优化我的选股标准" |
| "追求高覆盖率" | "接受≤10% 的极低通过率" |
| "让策略适配市场" | "让股票匹配我的系统" |
| "空仓=错过机会" | "空仓=纪律与选择" |
| "预设止盈止损控制风险" | "信号反转即卖，让利润奔跑" |

### 💡 关键认知

```python
# 错误的过拟合思维
for stock in all_stocks:
    params = optimize(stock)  # 为每只股票找"最优"
    if score(params) > threshold:
        trade()  # 任何股票都能做

# 正确的第一性原理
fixed_params = {macd=(12,26,9), exit_by_signal_reversal=True}
constraints = {min_return=0.10, min_sharpe=0.5, max_dd=0.25}

for stock in random_sample(10):  # 随机抽样，非刻意选择
    result = backtest(stock, fixed_params)
    if meets_constraints(result):
        trade()  # 只有符合规则的才做
    else:
        pass     # 不符合就放弃，空仓也是持仓
```

## ⚠️ 【重要】关于止盈止损的认知纠偏

### v2.1 重大更新（2026-03-27）

| 问题 | v2.0 的错误理解 | v2.1 的正确理解 |
|------|--------------|---------------|
| **为什么要设止盈？** | "我觉得涨 15% 该卖了" | ❌ 这是告诉市场只能赚多少 |
| **为什么要设止损？** | "我觉得跌 5% 受不了" | ❌ 这是主观猜测价格空间 |
| **什么时候卖出？** | 到价格点 or 死叉 | ✅ **买入信号消失即卖** |
| **收益率是谁决定的？** | 我设定的止盈点 | ✅ **市场给的反馈结果** |

### 🧠 更深层的第一性原理

```
预设止盈 = "教市场怎么走势" ← 这是人类后验标签
↓
信号驱动退出 = "尊重市场的真实走势" ← 这才是纪律

如果平均收益太低？
→ 收紧入场标准（选更高质量的标的）
→ 而不是调低止盈点（那不是解决问题的方法）
```

**收益是进化的反馈，不是预设的目标**。

## 🚨 重要提醒

### ❗ 预期管理

- **通过率会很低**：这是正常的，甚至是好事
- **不要试图提高通过率**：那是过拟合的信号
- **空仓是常态**：大部分时间等待，少数时间出击

### ⚠️ 风险提示

```
⚠️ 历史回测 ≠ 未来表现
⚠️ 过拟合风险：在更多样本上持续验证
⚠️ 实盘可能低于回测：留足安全边际
```
